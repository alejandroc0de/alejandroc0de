## Flight-Pomodoro

A unique Pomodoro technique, this app transforms focus sessions into simulated flights between airports.

Description:

Flight-Pomodoro is a productivity tool that matches Pomodoro session durations with real-world flight routes. Once a session starts, a plane travels from the origin to the selected destination, arriving exactly when the Pomodoro timer ends.

Tech Stack
- React
- JavaScript

Key Features

- Dynamic matching of Pomodoro time with flight routes
- Real-time plane animation during sessions
- Distance calculation using the Haversine formula
- Interactive and visual productivity experience
- Also a basic pomodoro if you're not into planes

Link : https://flight-pomodoro.netlify.app/focus
Repo : https://github.com/alejandroc0de/React/tree/4cc0bd7421a9a00fddd2fa7f5aac3d82a2509482/flight-pomodoro


## Thinkofyou

A real-time full-stack application designed for seamless communication.

Description

Thinkofyou is a real-time messaging platform that enables couples to communicate instantly through WebSockets, with a secure and scalable backend architecture. It sends a message to your partner to let them know  you thought of them

Tech Stack

- Node.js
- Express.js
- Socket.io (WebSockets)
- PostgreSQL
- JWT Authentication
- React

Key Features

- Real-time bi-directional communication
- Secure authentication with JWT
- Middleware-based security architecture
- Persistent data storage with PostgreSQL

Architecture

- WebSocket-based communication layer using Socket.io
- REST + middleware for authentication and request handling
